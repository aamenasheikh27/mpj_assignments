Mini Project Using Java- Assignments 

This repository contains solutions for the mini project using java  assignments.

# Assignment 1: Student Grade Application
- Takes student details and marks of 5 subjects
- Calculates average, percentage, and grade
- Concepts used: Class, constructor, methods, access specifiers

# Assignment 2: Employee Inheritance Application
- Demonstrates hierarchical inheritance
- Employee → FullTimeEmployee (50% hike)
- Employee → InternEmployee (25% hike)
- Concepts used: Inheritance, method overriding, polymorphism

  ## Assignment 3: Polymorphism in Java

- Demonstrates **method overloading** using Shapes class
- Uses **constructor overloading** for different shapes
- Menu-driven program to calculate area of:
  - Circle
  - Square
  - Rectangle
  - Triangle
- Demonstrates **method overriding (runtime polymorphism)** using Hillstations class and subclasses


## Assignment 4: File I/O and Exception Handling

- Menu-driven **Banking System**
- Uses **ObjectInputStream & ObjectOutputStream**
- Stores customer data in file (`customers.dat`)
- Implements:
  - Account creation
  - Deposit
  - Withdrawal
  - Display records
  - Delete account

### Exception Handling:
- MinimumBalanceException
- InsufficientFundsException
- InvalidCustomerIDException
- NegativeAmountException

## How to Run
1. Open the project in IntelliJ IDEA
2. Run:
   - `assignment1.StudentGradeApp`
   - `assignment2.EmployeeInheritanceApp`
